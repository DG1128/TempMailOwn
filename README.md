# Temp Mail Web App

A modern, random temporary email generator built with React, Vite, and TailwindCSS.

![Demo](https://via.placeholder.com/800x400?text=Temp+Mail+Web+App)

## Features
- **Custom Domains**: Select from multiple available domains.
- **Auto-Refresh**: Inbox updates every 2 seconds.
- **Privacy Focused**: Random usernames and secure HTML rendering.
- **Send & Receive**: Full email capabilities using mail.tm API.
- **Premium UI**: Dark mode interface.

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Technologies
- React 19
- Vite
- TailwindCSS v4
- Axios & SWR
- Lucide React Icons
